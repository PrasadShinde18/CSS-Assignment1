CSS can be added to HTML document in 3 way

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using the style attribute in the head section
  • External - by using a link element to link to an external CSS File.

When comparing CSS class vs ID the difference is that CSS class applies a style element.ID, On the other hand ,applies a style to one unique element.ID is all special in that you can use a special URL to link to directly to an element and its use by JavaScript.

In CSS, selectors are used to target the specific element or range range of elements on the webpages. Once an element has been targeted, a style or a set of style can be applied to the element.

There is wide range of selectoes available. Two of the most commonly used are class an ID. Both are used to target to which style should be applied.
